# Pipeline Color Rules

## Official Pattern:

### Done States (filled solid circles):
- **Submitted**: Filled blue circle (#1D4ED8)
- **Under Review**: Filled orange circle (#F59E0B)
- **Shortlisted**: Filled green circle (#059669)
- **Declined**: Filled red circle (#DC2626)

### Active State (white center + colored border + glow):
- **Current step only**: White center with colored border + 3px glow shadow
  - Under Review active: Orange border (#F59E0B)
  - Shortlisted active: Green border (#059669)
  - For Interview active: Green border (#059669)
  - Declined active: Red border (#DC2626)

### Inactive States:
- Gray circle (#F1F5F9) with gray border (#E2E8F0)

## Key Rule:
- **Done = Filled solid color**
- **Active = White center with colored border + glow**
- **Inactive = Gray**
- Done connector lines = Light blue (#BFDBFE)
